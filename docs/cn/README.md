# Demo-PrivateFile-iOS

## 介绍

这个项目演示如果在 MaxLeap 服务器上保存私有文件。

## 演示功能

- 用户注册和登录
- 加密上传文件
- 获取当前用户的加密文件列表
- 下载加密文件
- 删除加密文件

## 如何运行

- 在 leap.as 控制台中创建一个应用，下面称他为 MaxLeap 应用。如果已经创建，跳过这个步骤。
- 克隆这个仓库，然后打开项目
- 在 `AppDelegate.` 中填写 MaxLeap 应用的 applicationId 和 clientKey.
- 按下 Commond + R 按钮运行
